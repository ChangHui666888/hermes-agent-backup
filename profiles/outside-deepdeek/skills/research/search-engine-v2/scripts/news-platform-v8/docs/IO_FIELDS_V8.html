build_fingerprint() 内部结构

字段类型逻辑
subjectstrcompanies/persons 加权最大 (hub ×0.3, ≥0.15)
actionstr14 类枚举计数排序: ATTACKS|DIES|SUES|NEGOTIATES|...
objectstrcountries/companies 加权最高 (排除 subject)
event_typestrACTION_MAP[action][0]
primary_topicstrTOPIC_SIGNALS 命中最多
countrystr|Noneentities.countries[0] → location 硬约束
anchorstr{subject}|{action}|{object}|{topic} → 快速匹配

fingerprint_score() 评分

维度分数条件
location 硬约束return 0不同 country → 直接拒绝
anchor 快速通道100完全相同的 SAO+Topic → 立即合并
Action25同动作类型
Subject10-25同名 + 稀有度加权
Object10-30同名 + 稀有度加权
Topic10primary=10, secondary=5
阈值≥50 事件 | ≥75 强合并
